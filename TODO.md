# 开发路线图

本文档记录 Crayon Terminal 的待完成功能和未来计划。

## 高优先级

### 连接功能

- [ ] **SSH 隧道/端口转发**
  - 本地端口转发
  - 远程端口转发
  - 动态端口转发（SOCKS 代理）

- [ ] **SSH 跳板机支持**
  - ProxyJump 配置优化
  - 多跳连接可视化

- [ ] **SFTP 集成**
  - 文件浏览器
  - 拖拽上传/下载
  - 文件编辑

### 终端功能

- [ ] **终端录制**
  - 会话录制（asciinema 格式）
  - 录制回放

- [ ] **终端快照**
  - 保存终端状态
  - 恢复历史快照

- [ ] **智能补全**
  - 命令历史补全
  - 路径补全

## 中优先级

### 会话管理

- [ ] **会话同步**
  - 云端同步配置
  - 多设备同步

- [ ] **会话模板**
  - 预定义模板
  - 自定义模板

- [ ] **会话标签增强**
  - 标签颜色
  - 标签图标

### 界面优化

- [ ] **标签页拖拽排序**
  - 拖拽调整顺序
  - 拖拽到新窗口

- [ ] **终端透明度**
  - 背景透明度调节
  - 毛玻璃效果

- [ ] **状态栏增强**
  - CPU/内存监控
  - 网络流量显示

### 命令库

- [ ] **命令分类**
  - 按标签分类
  - 按服务器分类

- [ ] **命令历史**
  - 执行历史记录
  - 快速重复执行

- [ ] **命令片段**
  - 代码片段管理
  - 快速插入

## 低优先级

### 脚本功能

- [ ] **脚本引擎**
  - Lua/JavaScript 脚本
  - 自动化任务

- [ ] **定时任务**
  - 定时执行命令
  - 定时备份

### 其他功能

- [ ] **快捷键录制**
  - 宏录制
  - 按键序列回放

- [ ] **多窗口支持**
  - 独立窗口
  - 窗口管理

- [ ] **插件系统**
  - 插件 API
  - 第三方插件支持

- [ ] **远程会话共享**
  - 会话共享链接
  - 协作模式

## 贡献指南

如果您想贡献代码，请：

1. Fork 项目
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request
